# AI Usage Notes

## Where AI Was Used

- Structured prompt design for generating user stories, engineering tasks, and risks.
- Integration guidance for Groq SDK.
- Debugging backend errors (model deprecation, environment issues).
- UI improvements and layout suggestions.
- Assistance writing deployment and documentation files.

- Prisma schema and database connectivity.
- Response parsing logic.
- Error handling.
- Download functionality.
- Deployment configuration.

## LLM Used

Provider: Groq  
Model: llama-3.1-8b-instant  

Reason:
- Fast inference
- Stable structured output
- Good balance of cost and quality
- Simple SDK integration

Groq was chosen for low latency and clean production deployment.
